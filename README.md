
### 项目简介

一款类似《胡闹厨房》的游戏。

**实现功能**：
- 场景切换
- 玩家与操作台的交互：拾取放置食物，烹饪食物，用盘子盛装食物，提交食物
- 玩家移动
- 待提交食谱的UI变化，提交成功或失败的动画UI显示
- 简易的按钮绑定切换
- 在选项菜单中调整音量大小并实现数据持久化
